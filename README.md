#  Stopwatch Web Application  

A modern and responsive stopwatch web application built using **HTML, CSS, and JavaScript**.  
This project provides accurate time tracking with lap recording and a clean professional UI.

---

##  Features  

- ▶️ Start stopwatch  
- ⏸️ Pause stopwatch  
- 🔄 Reset stopwatch  
- 🏁 Record lap times  
- 🗑️ Clear all laps  
- 📊 Lap counter with latest lap displayed first  
- 📱 Responsive design for desktop and mobile  

---

##  Technologies Used  

- HTML5  
- CSS3  
- JavaScript (Vanilla JS)
